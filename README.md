# Title
“Heart Disease Risk Prediction from Clinical Data”
## Team members
Nyla Young (nylayoung), Charles Clark (GitHubUserB)
## Project decription
This project will focus on predicting the likelihood of heart disease using basic clinical data. Heart disease is one of the leading causes of death worldwide, and early risk prediction can help to guide preventative care. In this project we willl use structured clinical data which makes it manageable and medically relevant.

We plan to use a publicly available dataset which is the Cleveland Heart Disease dataset from Kaggle. This dataset includes variables like age, sex, blood pressure, cholesterol, maximum heart rate, and other clinical measurements that may be associated with heart disease.

The goal of the project is to create and train a machine learning model that can classify whether a patient is at risk for heart disases. We may compare models such as logistic regression, decision trees, random forests, or a simple nearual network. Model performance will be evaluated using measures such as accuracy, precision, recall, and confusion matrices.
